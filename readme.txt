Instructions: 
1. Choose a segment name from the segment drop down list (NAM or BIO)
2. Choose a field name from the field drop down list (FNA, LNA, DOB)
3. If you choose a NAM, you can either choose FNA or LNA and the text will populate with the first or last name of that segment name
4. If you choose BIO, the text will populate with the DOB or date of birth of that segment name
5. The text will appear in the text box area
